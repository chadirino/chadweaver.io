# chadweaver.io

Personal portfolio website for **Chad Weaver** — a Data Analyst and Civic Tech builder based in Chicago, IL. This site serves as a central hub for my projects, professional history, and experiments in civic technology.

## 🚀 Live Site
[chadweaver.io](https://chadweaver.io)

---

## 🛠 Tech Stack
- **Framework:** None — plain HTML, CSS, and JavaScript
- **Styling:** Vanilla CSS with custom properties (mobile-first)
- **Icons:** Lucide (SVG, inlined)
- **Animations:** Intersection Observer API
- **Deployment:** GitHub Pages

## 🏗 Local Development

1. **Clone the repo:**
   ```bash
   git clone https://github.com/chadirino/chadweaver.io.git
   ```

2. **Open locally:** No build step required. Open any `.html` file directly in a browser, or serve with any static file server:
   ```bash
   npx serve .
   ```
